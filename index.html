¿Cómo instalar Bootstrap mediante NPM de manera local?

Paso 1

Abrir Visual Studio Code para inicializar un proyecto.
Abrir la terminal del VSCode y ejecutar "npm init -y".
Ejecutar "npm install bootstrap@5.3.3" en la consola.

Paso 2

Creamos el "index.html" en la carpeta raíz del proyecto.
Creamos un archivo llamado "styles.scss".
Dentro del archivo copiamos @import "node_modules/bootstrap/scss/bootstrap.scss".

Paso 3

Prepararemos el CSS, para ello instalar la extensión "Live Sass Compiler".
En la parte inferior derecha encontraremos un bóton "Watch Sass", damos click en él.
Volvemos al archivo style.scss, hacemos cualquier cambio y guardamos el archivo (eso compilará el archivo).

Paso 4

Volvemos al "index.html".
Vínculamos la hója de estilo creada con:
"link rel="stylesheet" href="style.css"", en el head.
Vínculamos la hoja de scrips con:
"script src="node_modules/bootstrap/dist/js/bootstrap.min.js"", al final del body.
